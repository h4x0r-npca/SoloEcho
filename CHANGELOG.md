# Changelog

SoloEcho는 Git 태그와 이 변경 이력을 함께 사용해 버전을 관리합니다.

버전 표기는 `MAJOR.MINOR.PATCH+BUILD` 형식을 따릅니다.

- `MAJOR`: 공개 배포 기준의 큰 구조 변화
- `MINOR`: 사용자에게 보이는 주요 기능 추가
- `PATCH`: 보완, 안정화, 작은 기능 추가
- `BUILD`: 앱 스토어/플랫폼 빌드 번호용 정수

GitHub에서는 각 버전에 `v0.2.1`처럼 태그를 붙입니다. 새 버전을 낼 때는 `pubspec.yaml`의 `version` 값을 올리고, 이 파일에 변경사항을 적은 뒤, 커밋과 태그를 함께 푸시합니다.

## 0.3.0+4 - 2026-07-02

- 설정에서 켜고 끌 수 있는 로컬 잠금모드를 추가했습니다.
- 잠금 비밀번호는 평문 저장 없이 솔트가 포함된 PBKDF2-SHA256 해시로 저장합니다.
- 앱 시작과 백그라운드 복귀 시 잠금화면을 표시하도록 했습니다.
- 잠금화면은 기존 로그인 화면의 파스텔 배경 애니메이션을 재사용합니다.
- 설정에서 화면 전환/저장 애니메이션을 켜고 끌 수 있게 했습니다.
- 로그인/잠금/홈 화면 전환을 동일한 디졸브 효과로 통일하고 전환 시간을 늘렸습니다.
- 채팅방식 저장 성공 시 새 말풍선이 아래에서 위로 올라오는 효과를 추가했습니다.
- 스레드방식 저장 성공 시 최신 글 위치로 자연스럽게 스크롤되는 효과를 추가했습니다.
- 설정 화면의 계정 아바타도 Google 프로필 사진을 사용하도록 통일했습니다.
- Windows/macOS 데스크톱 앱의 초기 창 크기를 세로형으로 맞췄습니다.
- Android 시작/복귀 시 기본 템플릿 색이 비치지 않도록 네이티브 창 배경과 앱 기본 배경을 보강했습니다.

## 0.2.1+3 - 2026-07-01

- 색상 테마 전환 기능을 추가했습니다.
- 로그인 이후 화면의 글씨 크기 조절 기능을 추가했습니다.
- 채팅방식과 스레드방식에서 검색 기능을 추가했습니다.
- 스레드방식에서 `Ctrl + Enter`로 기록할 수 있게 했습니다.
- macOS 테스트 패키징 스크립트를 추가하고, 앱 네트워크 권한이 빠지지 않도록 서명 검증을 보강했습니다.
- macOS 테스트 앱에서 네트워크 인증 문제가 생길 때 한국어 안내 문구를 표시하도록 했습니다.

## 0.2.0+2 - 2026-07-01

- SoloEcho 로고를 앱 아이콘, 로그인 화면, 메인 화면에 적용했습니다.
- 로그인 화면에 감정 변화 느낌의 파스텔 배경 애니메이션을 추가했습니다.
- Android, iOS, macOS, Windows에서 로고 리소스를 사용할 수 있도록 정리했습니다.
